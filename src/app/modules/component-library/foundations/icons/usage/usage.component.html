Status Indicators Guidelines

Status indicators are color coded icons that allow the user to scan a table and quickly see the state of the items in that table. While the wording of these statuses may vary, we follow specific rules for the color and shape of the status indicator. Status indicators must always have a label for accessibility purposes.
