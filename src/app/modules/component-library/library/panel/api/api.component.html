Inputs

Panel inputs
Input Type Description Notes
titleText string The text used for the panel's title.
panelHeaderLink boolean Determines if the panel header contains a content projection area for a link.

Default is "false".

When "true" the panel header contains a content projection area for a link.
panelFooter boolean Determines if the panel contains a content projection area for a footer.

Default is "false".

When "true" the panel contains a content projection area for a footer.

Content Projection

The optional ng-content directives' select attribute values are panel-header-link, panel-body, and panel-footer.
