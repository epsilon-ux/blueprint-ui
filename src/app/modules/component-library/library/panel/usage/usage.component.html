Panels contain a title and body content. They have no set width and can be adjusted depending on the content.

Panels usually contain charts or tables in the body. They also can have links in the header or footer that take the user to see more information related to the panel content.

Variations

Standard panel

This is the standard panel. Use this variation for a panel that does not need a primary or secondary link.

Content projection area for panel body

Panel with primary link

Panels can have a primary link that appears in the tile header. The link is usually a ‘view more’ link.

This is a panel using content projection to add a link in the panel header

Panel with secondary link

Panels can have a secondary link that appears in the panel footer. Use this variation when the link should be de-emphasized or pertains more to the content at the bottom of the content area.

This is a panel using content projection to add a panel footer

Panel with primary link and secondary link

Panels can have primary and secondary links. Use this variation when there are two different places the user could go to view more information related to the panel content.

This is a panel using content projection to add a link in the panel header and a panel footer
