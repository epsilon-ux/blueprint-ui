Inputs

Radio button inputs
Input Type Description Notes
label string The radio button's label. Required.
id string The value for the radio button's id attribute.

Ids must only be used once per page.

Ids should be camelCase, e.g.: #myId.
name string The value for the radio button's name attribute.

When a form is submitted, the name attribute's value is the reported name of the radio button if the radio button is checked and enabled.

For a group of radio buttons, the name attributes' values should be the same.
value string The value for the radio button's value attribute.

Default is "on".

When a form is submitted, the value attribute's value is the reported value of the radio button if the radio button is checked and enabled.

For a group of radio buttons, the value attributes' values should be unique.
isChecked boolean Determines if the radio button is checked or unchecked.

Default is "false".

When "true" the radio button is checked.

When a form is submitted, a radio button's value is only included in the submitted data if the radio button is checked and enabled.
isDisabled boolean Determines if the radio button is disabled or enabled.

Default is "false".

When "true" the radio button is disabled.

When a form is submitted, a radio button's value is only included in the submitted data if the radio button is checked and enabled.
isRequired boolean Determines if the radio button is required or optional.

Default is "false".

When "true" the radio button is required.
