Selector
bp-checkbox

Inputs

Checkbox inputs
Input Type Description Notes
label string The checkbox's label. Required.
visuallyHiddenLabel boolean Determines if the checkbox's label is visible or visually hidden.

Default is "false".

When "true" the label is visually hidden.
bpID string The value for the checkbox's id attribute.

ids must only be used once per page.

ids should be camelCase, e.g.: #myId.

See identifiers.
name string The value for the checkbox's name attribute.

When a form is submitted, the name attribute's value is the reported name of the checkbox if the checkbox is checked and enabled.

If a group of checkboxes are unrelated, the name attributes' values should be unique.

If a group of checkboxes are related, the name attributes' values should be the same.
value string The value for the checkbox's value attribute.

Default is "on".

When a form is submitted, the value attribute's value is the reported value of the checkbox if the checkbox is checked and enabled.
isChecked boolean Determines if the checkbox is checked or unchecked.

Default is "false".

When "true" the checkbox is checked.

When a form is submitted, a checkbox's value is only included in the submitted data if the checkbox is checked and enabled.
isDisabled boolean Determines if the checkbox is disabled or enabled.

Default is "false".

When "true" the checkbox is disabled.

When a form is submitted, a checkbox's value is only included in the submitted data if the checkbox is checked and enabled.
isRequired boolean Determines if the checkbox is required or optional.

Default is "false".

When "true" the checkbox is required.
