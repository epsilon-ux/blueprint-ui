Inputs

Landing page header inputs
Input Type Description Notes
title string The title text at the top of the secondary navigation.
items NavItem[] The object structure that creates the navigation items. The nesting of the NavItem objects will reflect as nesting in the navigation structure.
shouldRouteMatchExact boolean Determines if the routes should match exactly to be active. Equates to the routerLinkActiveOptions “exact” option.

Default is "false".

When "true" the routes should match exactly.
isNavCollapsed boolean Determines if the secondary navigation should be open or closed.

Default is "false".

When "true" the nav is collapsed.
areItemsExpanded boolean Determines if the nested navigation items should be expanded.

Default is "false".

When "true" the nested navigation items are all expanded.
