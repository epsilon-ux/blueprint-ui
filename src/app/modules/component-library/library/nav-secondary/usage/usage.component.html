The left panel contains secondary navigation. The side panel appears under the top navigation and can be collapsed or expanded. The secondary navigation usually navigates within an object.

Variations

Standard secondary navigation

This is the most commonly used configuration. Use this when you don't need a panel title or page groups.

Main Content

Secondary navigation with title

Use this configuration when the secondary navigation needs a panel title. Titles are used when the panel content is related, but does not interact with each other or belong to a single object or detail page. An example of this is Business Unit Settings in PeopleCloud Messaging.

Main Content

Secondary navigation with page groups

Use page groups when you need to nest pages within a parent category. This cuts down on the number of pages shown and organizes the panel into an scannable list. Single pages and page groups can exist in the same panel.

Configurations

Secondary navigation optional elements and descriptions
Element Purpose Notes
Title Categorizes the collection of navigation items. Use when the navigation items don't belong to one object, but are still related and belong in the same category.
Page groups Nests child pages within parent groups. Use when navigation items should appear in a single group or belongs within the same category.
