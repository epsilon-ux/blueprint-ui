Selector
bp-pagination

Inputs

Pagination inputs
Input Type Description Notes
dataLength number The length of the data that is being paginated. Required.
defaultNumberOfRows number The amount of data to be shown on component load.

Required.

Corresponds to the optionValues of the "number of rows" selector.
pageBuffer number The number of pager buttons to show in the page selector before and after the current page.

Default is "1".

When "1", there will be one page number on either side of the current pager button. On the first or last page there will be two pager buttons to either the right or left of the current pager button respectively.
pageNumber number Manually sets the page number.
internationalization object Used to change any text in the component and set the option vales of the "number of rows" selector. See internationalization.

Internationalization

Internationalization
Input Type Description Notes
'numResults results' string The text for the number of results indicator. Default is "#{numResults} result(s)"
'Number of Rows' string The for the aria label on the "number of rows" selector. Default is "Number of Rows per Page"
'First Page' string The aria label on the "first page" button. Default is "First Page"
'Previous Page' string The aria label on the "previous page" button. Default is "Previous Page"
'Next Page' string The aria label on the "next page" button. Default is "Next Page"
'Last Page' string The aria label on the "last page" button. Default is "Last Page"
'Number of Rows Options' object[] The option text and values to use in the "number of rows" selector. The object should have two key value pairs. The keys are optionText and optionValue.

Outputs

pageData

pageData object details
Property Description
currentPage A number revealing the current page.
rowsPerPage The current value selected in the "number of rows" selector.
indices An object containing start and end keys whose values correspond to the start and end indices of the data based on the current page and number of rows option selected.
