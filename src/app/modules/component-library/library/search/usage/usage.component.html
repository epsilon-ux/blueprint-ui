The search component is most often used on pages that contain data tables.

Configurations

All of the elements in the search component are required. The elements in the table below are configurable.

Search configurable elements and descriptions
Element Purpose Notes
Label Informs the user what information to enter into its corresponding form control.

The label text should indicate what can be searched.

When used in combination with a data table, the label text should be specific if the search is performed on one column, e.g.: "Search Name", or generic if the search is performed on multiple columns, e.g.: "Search".

Placeholder Provides the user with a short hint intended to aid with data entry when the control has no value. The placeholder attribute is not a substitute for a label.
