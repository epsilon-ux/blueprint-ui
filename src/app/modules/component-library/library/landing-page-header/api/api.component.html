Inputs

Landing page header inputs
Input Type Description Notes
headingText string The landing page's title. Required.
actionText string The action's text. Required when action is present e.g. routerLink/href input or actionClick output is provided.
dropdownItems object[] An array of objects used to create the items in the dropdown.
routerLink string The action's routerLink attribute value. Cannot have both routerLink and href.
href string The action's href attribute value. Cannot have both routerLink and href.

Outputs

Landing page header outputs
Output Description
actionClick Emits an event on click of the primary button or dropdown menu.

Content Projection

The dropdown-items and tab-items attributes are values for the optional ng-content directives' select attribute.
