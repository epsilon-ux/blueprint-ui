Inputs

Landing page header inputs
Input Type Description Notes
headingText string The landing page's title.

Required.
hasAction boolean Determines whether or not there is a primary button or dropdown menu.

Expects "true" or "false".

Default is "false".

Use "true" to render a primary button or dropdown menu.
actionText string The action's text. Required if hasAction is "true".
actionElement string Determines what element and/or attributes to use for the action.

Expects "a", "button", or "dropdown".

Default is "a".

Use "a" when the action takes the user to another page.

Use "button" when the action peforms an action on the current page, e.g.: opens a modal.

Use "dropdown" when there are multiple actions.
href string The action's href attribute value. Required if actionElement is "a".

Outputs

Landing page header outputs
Output Description
actionClick Emits an event on click of the primary button or dropdown menu.
