Selector
bp-progress-bar

Inputs

Progress bar inputs
Input Type Description Notes
min string The starting point of the progress bar (0%). Required.
max string The ending point of the progress bar (100%). Required.
value string The current progress (0%-100%). Required.
classes string The classes applied to the progress bar. See the Configurations table on the Usage page to determine the classes needed.
hasProgressText boolean Determines if a textual representation of the progress is rendered below the progress bar.

Default is "false".

When "true" a textual representation of the progress renders below the progress bar.
internationalization object Allows the text used in the component to be changed. See Internationalization.
internaitonalization properties
Input Type Description Notes
'% complete' string The text used below the progress bar when the Input hasProgressText is true.

Default is '#{value}% complete'.

Use '#{value}' to place the percentage value within the string.
