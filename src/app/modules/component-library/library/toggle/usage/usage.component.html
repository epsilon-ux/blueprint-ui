Toggles are stylized radio buttons, and function like a radio group. There must be a default selection and the user can only select one option. Toggles should deliver an immediate result when the user switches to another option.

Toggles should be used for binary or mutually exclusive options and should activate or deactivate a state. Do not use toggles for navigation or actions. Limit the use of toggles within forms, unless the toggle has two binary options, such as activating or deactivating.
