Label and optionItems are the only requried Inputs for the multiselect component. All other additions shown below are optional and depend on the situation.

Variations

Standard multiselect

This is a barebones variation of the multiselect.

Standard multiselect with search

Use this variation to allow searching in a multiselect.

Standard multiselect with disabled options

Use this variation to disable individual options within the select menu.

Disabled multiselect

Use this variation when the user can not change the value of the multiselect, and the selected items should not be submitted in a form.

Readonly multiselect

Use this variation when the user can not change the value of the multiselect, and the selected items should be submitted in a form.

Standard multiselect with custom placeholder

Use this variation when the user can not change the value of the multiselect, and the selected items should be submitted in a form.
