Object headers vary from product to product depending on what information needs to show in the header.

Information options are shown below the object name. These options can be turned on or off depending on the object type. Object headers can have up to three buttons before overflowing to a 'More Actions' dropdown button.

In most products, the user edits the object name using an object name input field. This field must be the first component in the object hierarchy.

The object page header appears below the primary navigation and to the right of the secondary navigation. The secondary navigation is optional.
