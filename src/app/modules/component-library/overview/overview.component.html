Setup & Installation

This package has a few dependencies that are not included in the package itself. They are as follows:

  • Angular 8
  • Ng-select
  • Bootstrap v4
  • Font Awesome 5

Once you have all these, run npm install epsilon-blueprint to install the package to your node_modules. Once the package is installed you then need to import the stylesheets from the package.

In your application styles.scss file, add the following import statement. @import "~epsilon-blueprint/styles";

When you need to use global variables in a scss file, add the following import statement. @import "~epsilon-blueprint/styles/global-variables";
