Selector
bp-action

Inputs

The action component will render either an anchor or a button depending on the inputs. If either an href or bpRouterLink is provided, the action will be an anchor. If neither are provided, the action will be a button.

Action inputs
Input Type Description Notes
text string The text used in the element.
bpType string The value of the button's type attribute. Only valid when the action is a button.
classes string The classes to apply to the element. See the configurations table in Usage to determine the classes needed.
ariaLabel string The text used as the aria label of the element.
href string The value for the action link's href attribute. Action cannot have both href and bpRouterLink.
bpRouterLink string The value for the action link's routerLink attribute. Action cannot have both href and bpRouterLink.
target string The target for the link. For example, "_blank" opens in a new tab.
isDisabled boolean Determines if the button is disabled or enabled.

Default is "false".

When "true" the button is disabled.
iconLeft string The Font Awesome icon class for an icon on the left of the element. For example, "fa-external-link-alt" for an external link icon.
iconRight string The Font Awesome icon class for an icon on the right of the element. For example, "fa-chevron-left" for a left facing chevron icon i.e. back link.
bpID string The value for the action's id attribute.

Ids must only be used once per page.

Ids should be camelCase, e.g.: #myId.

See identifiers.

Outputs

Action outputs
Output Description
click Emits an event on click of the action, if the action is a button.

Identifiers

Identifiers
