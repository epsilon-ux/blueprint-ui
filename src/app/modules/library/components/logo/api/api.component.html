Inputs

Badge inputs
Input Type Description Notes
product string The product logo to show.

Default is "product".

Can be one of the following: "product", "blueprint", "customer", "data-hub", "events", "iq", "loyalty", "messaging"

The logo's title will be set based on the product specified.

logoTitle string Overrides the default title determined by the product Input.
bpClass string Passes classes down to the svg element.
