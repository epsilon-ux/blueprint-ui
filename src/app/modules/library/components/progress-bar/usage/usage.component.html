Progress bars are used to visualize the progression of a task or operation. Progress bars are often accompanied by a textual representation of the progress in a percent format.

Variations

Standard progress bar

The standard progress bar is used on dashboards and data visualization pages.

Slim progress bar

The slim variation should be used when vertical space is limited, like in table cells.

Success progress bar

This is used for any positive progression that calls for a different style than the standard progress bar.

Slim success progress bar with progress text

This is used for any positive progression that calls for a different style than the standard progress bar when vertical space is limited, like in table cells, and when the progress bar should be accompanied by a textual representation of the progress.
