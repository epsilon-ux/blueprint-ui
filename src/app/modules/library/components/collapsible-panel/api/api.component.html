Selector
bp-collapsible-panel

Inputs

Collapsible panel inputs
Input Type Description Notes
isPanelCollapsed boolean Determines if the panel is collapsed.

Default is "false".

When "true" the panel is collapsed.
toggleAriaLabel string The open/close button's aria label. Default is "Toggle collapsible panel".
isPanelRight boolean Determines if the panel renders on the right side of the viewport.

Default is false.

When "true" the panel is rendered on the right side of the viewport.
bpID string The value for the collapsible panel's id attribute.

Ids must only be used once per page.

Ids should be camelCase, e.g.: #myId.

See identifiers.

Content Projection

The optional ng-content directives' select attribute value is panel-content.

panel-content

A content area in the collapsible panel used for filters, secondary navigation, tabs, or static content.

Identifiers

Identifiers
ID Description
bpID The unique identifier for the collapsible panel component.
bpID + 'Toggle' The unique identifier for the collapsible panel's toggle button.
