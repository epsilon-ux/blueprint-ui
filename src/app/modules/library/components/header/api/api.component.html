Selector
bp-header

Inputs

Header inputs
Input Type Description Notes
product string The product using the header. Used to determine the logo.

Default is "product".

Can be one of the following: "product", "blueprint", "customer", "data-hub", "events", "iq", "loyalty", "messaging".

The logo's title will be set based on the product specified.

logoTitle string Overrides the default title determined by the product Input.
hasCollapsibleMenu boolean Determines if the header has a collapsible menu. The collapsible menu contains everything but the logo and becomes collapsible at certain breakpoints.

Default is "true".

When "true" there is a collapsible menu.
hasContextSelecotr boolean Determines if the header has a context selector.

Default is "true".

When "true" the header has a context selector.
hasNavigation boolean Determines if the header has navigation. Navigation includes a major and minor section.

Default is "true".

When "true" the header has navigation.
expandableBreakpoint string The breakpoint at which the collapsibled menu becomes non-collapsible.

Default is "lg".

Uses Bootstrap's breakpoints.
bpID string Override the default base id. See identifiers.
internationalization object Allows any user facing text to be changed. See internationalization.

Internationalization

Internationalization inputs
Input Type Description Notes
'Menu' string The text used in the collapsible menu trigger. Default is "Menu".
'MenuAriaLabel' string The text used for the collapsible menu trigger's aria-label. Default is "Toggle primary navigation".
'NavAriaLabel' string The text used for the nav's aria-label. Default is "Primary Navigation".

Identifiers

Identifiers
ID Description

bpID + 'Menu' The unique identifier for the collapsible menu.
