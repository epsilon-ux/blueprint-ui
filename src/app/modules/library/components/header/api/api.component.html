Selector
bp-header

Inputs

Header inputs
Input Type Description Notes
text string The badge's text.
element string The element used for the badge.

Expects either "span" or "div".

Default is "span".

Use "div" to create a block-level badge.
classes string The classes applied to the badge.

Default is "badge-primary".

See the Configurations table on the Usage page to determine the classes needed.
