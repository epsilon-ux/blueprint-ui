Selector
bp-loading-indicator

Inputs

Loading indicator inputs
Input Type Description Notes
isLoading boolean Determines if the loading indicator is rendered.

Default is "false".

When "true" the loading indicator is rendered.
Inline/In-Page/Over-Page Determines if the loading indicator is rendered inline, in-page, or over-page.

Default is one of these.

When "inline" the loading indicator renders like an inline element.

When "in-page" the loading indicator renders as part of a page.

When "over-page" the loading indicator renders on top of the application's UI and blocks any user interactions.
