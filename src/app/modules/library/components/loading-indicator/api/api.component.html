Selector
bp-loading-indicator

Inputs

Loading indicator inputs
Input Type Description Notes
text string The text to display with the loading indicator. Must provide either text or ariaLabel.
ariaLabel string The value of the loading indicator's aria-label. Must provide either text or ariaLabel.
kind string Determines if the loading indicator is rendered inline, in-page, or over-page.

Can be 'inline', 'inPage', or 'overPage'.

When "inline" the loading indicator renders like an inline element.

When "inPage" the loading indicator renders in the relevant section of a page.

When "overPage" the loading indicator renders on top of the application's UI.
