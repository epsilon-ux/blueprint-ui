Radio buttons are used for lists of two or more options that are exclusive of each other.

The user must make a selection from the list and can only select one option. When a user selects an option, the previously selected option is automatically unselected. The first option in a radio button set should always be selected by default. If the first option is disabled, the second option should be selected.

An unselected radio button represents a negative selection and a selected radio button represents a positive selection. If there is a large number of options and not enough space, consider using a dropdown.
