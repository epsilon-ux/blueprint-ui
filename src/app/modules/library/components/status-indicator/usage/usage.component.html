Status indicators often appear in tables and object page headers, but can also appear in other contexts, like an object's status page. While the wording of these statuses may vary, we follow specific rules for the color and shape of the status indicator. Status indicators must always have a label for accessibility purposes.

Variations

Incomplete primary status indicator

The incomplete primary status indicator represents that an action is not done and the user needs to take an action.

Incomplete secondary status indicator

The incomplete secondary status indicator represents that an action is not done and the user needs to take an action. This status should only be used if the 'incomplete primary' status has already been used to indicate a different status.

In progress status indicator

The in progress status indicator represents that an action is in progress. The spinner icon is always animated.

Complete status indicator

The complete status indicator represents that an action is done and there is no further action needed.

Warning status indicator

The warning status indicator represents that an action resulted in an error.
