Selector
bp-chart-line

Inputs

Line Chart inputs
Input Type Description Notes
data object[] The data used to create the chart. If the data needs to be updated dynamically, create a copy of the data, make the needed changes, then reassign the new array to the input variable. Otherwise Angular will not pick up on the change detection.
groupBy string The key used to group the data. This groups te data and creates a line for each group. If not provided, a single line will be drawn.
xValue string The key from the data to use for the x values. Required.
yValue string The key from the data to use for the y values. Required.
xAxisLabel string The text used for the y axis label.
yAxisLabel string The text used for the x axis label.
bpID string The value for the SVG's id attribute.

Ids must only be used once per page.

Ids should be camelCase, e.g.: #myId.

See identifiers.
title string The string to use as the SVG's title element. Required.
formatters object An object containing formatter functions for the information in the pie chart. See formatters.

Identifiers

Formatters
Property Type Description Notes
xAxis (date) => string The formatter function for the x axis ticks.
yAxis (number) => string The formatter function for the y axis ticks. Default is (value: number): string => new Intl.NumberFormat().format(value).
tooltipHeader (any) => string The formatter function for the tooltip header text. Default is (value: any): string => new Intl.DateTimeFormat().format(value).
tooltipLabel (any) => string The formatter function for the tooltip label text. Default is (value: any): string => String(value).
tooltipValue (number) => string The formatter function for the tooltip value text. Default is (value: number): string => new Intl.NumberFormat().format(value).
